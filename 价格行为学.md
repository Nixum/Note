# 一、止损

交易员的四项技能：

1. 知道市场在做什么，接下来怎么做，即市场周期，趋势强弱，是否反转，震荡宽还是窄
2. 入场时机
3. 仓位管理：目标位（止盈或止跌）、以来初始止损 or 提前离场、窄止损 or 宽止损、打平 or 小亏、加仓
4. 情绪

新手练习：做波段，入场 -》设置目标位，果断执行

止损=救命，**本金才是最重要**，市场一直在那，重要的是要有入场的资格，尽量控制每只股票亏损在本金的2%以内

## 上涨趋势的止损

* 高、低点不断抬高，尤其是低点，低点的抬高更加重要

* 止损放在重要的低点下方

* 分辨好 重要的低点 和 次要的低点
  * 上涨趋势中的重要低点：强势突破创新高；
  * 止损点最好不要放在次要低点下方，因为市场有可能把止损打掉，继续上涨，次要低点说明不了什么
  * **跌破重要低点后，就要开始判断市场是否处于上涨趋势，还是已经进入震荡区间，甚至会反转为下跌**

**尽管跌破重要低点后，仍然可能大涨，但是已经止损离场，就不要觉得可惜了，因为那也是之后的事，在当下它就是最正确的选择。要记住，不存在完美的止损，而是尽量保住本金，寻找下一次机会就行，不需要每次都很完美**，另外，入场离场，需要的是判断盈亏比，如果亏损几率过高，或者五五开，还不如不做。

![](https://github.com/Nixum/Java-Note/raw/master/picture/上涨趋势-低点不断抬高.png)

## 下跌趋势的止损

* 跟上涨趋势的止损相反
* 强下降趋势往往会无视很多支撑位，但是下降趋势停止的地方，一定是某个支撑位
* 当遇到有连续的下跌k线，就可以判断Measured Move
* 下跌后反弹，需要判断缺口（缺口就是反弹后的最高点是否达到上一次的低点），实体缺口是指收盘价之间的缺口，有缺口说明下跌趋势的概率很大，还没止跌，达到Measured Move的概率越大

## 反转趋势下的止损

* 急速变通道（弱趋势），通道变区间
* 判断阻力位，比如MM，结合EMA20
* 是否有趋势的多推（多个波动峰谷）

![](https://github.com/Nixum/Java-Note/raw/master/picture/上涨反转.png)

## 看跌期权止损

* 最适合买入看跌期权的时机：在上涨趋势中，波动特别小，但是仍然再上涨，这个时候买入看跌期权进行保护非常合适，因为此时 IV 比较小，看跌期权的价格会比较便宜，适合买入看跌期权进行保护，而且就算一直上涨，看跌期权的损失也比较小。另外，客观来说，上涨幅度都很小，但是下跌幅度可能会很大，如果遇到大幅下跌的情况，看跌期权就非常有用了，此时可以卖掉看跌期权赚一笔，或者行权进行保护，当然前提是行权价高于你的买入价，此时的买入的看跌期权就能以最小的成本获得最大的对冲效果

* 一般来说公司发财报前，IV都是非常高的，此时看跌期权可能非常贵，最优的方式是做sell covered call，再用sell call赚来的钱，去买put，以此减少损失。
* IV 高的情况下，一般不买看涨/跌期权，因为可能对应的概率不高，但是因为是买入，价格会很贵

## 防止大亏的入场规则

1. 强势信号k线 + 突破单入场 + 保护性止损
2. 第二入场点入场 、 第二次反转时离开
3. 不加仓
4. 风险 & 仓位控制
5. 亏损之后等待第几根k线再做决定
6. 避开重要事件

# 二、仓位管理

1. 每一笔交易的风险不超过账户的2%，包括加仓 或 宽止损
2. 如果追涨第一次买入时，离下方的低点比较远，为了控制仓位，则可以考虑 2% * 20%的小仓位买入
3. 价格行为决定止损，止损决定仓位
4. 不要频繁地加大仓位

# 四、图例讲解

## 上涨4种常见的趋势

![](https://github.com/Nixum/Java-Note/raw/master/picture/4种常见趋势.png)

* 突破：三根大阳线，做多，因为强势的上涨胜率高，买入有较大的概率盈利，强突破往往伴随着两端连续突破，就算第一次回调了，第二次上涨的概率也比较大，可以加仓，但是风险也相对较高；

* 窄通道：幅度小，回调的k线数量少，比如只有3根以内

* 宽通道：伴随多次回调，每次5根以上，可能有多个推，一般也不要在第三推追涨，持续时间长，只能快速止盈；

  在宽通道里的50%回调是关键，这样延续左边的趋势的概率还是比较大，可以测试高点，如果回调至三分之一的位置或者前期低点，说明后续走入震荡区间的概率大；

* 震荡区间：向上突破失败，向下突破失败，更需要快速止盈，更频繁的高抛低吸

# 五、概念

## Measured Move

可译为**目标位测量**或**等距运动**：用于预测价格目标。它的核心理念是：市场在完成一次趋势行情和一次修正行情后（回调），很可能会再次发起一段与**第一段趋势行情幅度相等**的走势。

Measured Move的典型结构：一个标准的Measured Move由三个清晰的阶段构成，形成一种“推进-修正-再推进”的节奏。

| 阶段                  | 牛市中的表现 (看涨)                                      | 熊市中的表现 (看跌)                                    | 核心作用                                      |
| :-------------------- | :------------------------------------------------------- | :----------------------------------------------------- | :-------------------------------------------- |
| **1. 初始推动 (AB)**  | 价格从低点A强势上涨至高点B。                             | 价格从高点A强势下跌至低点B。                           | 确立趋势方向和**基准测量距离**。              |
| **2. 修正/整理 (BC)** | 价格从B点回落/横盘至C点（通常回撤AB段涨幅的30%-62%）。   | 价格从B点反弹/横盘至C点（通常回撤AB段跌幅的30%-62%）。 | 消耗反向能量，为下一波行情蓄力。              |
| **3. 目标推动 (CD)**  | 价格从C点再次上涨，目标位D = 在C点上**加上AB段的距离**。 | 价格从C点再次下跌，目标位D = 从C点**减去AB段的距离**。 | 提供可量化的**盈利目标**（即Measured Move）。 |





# 六、复盘

![](https://github.com/Nixum/Java-Note/raw/master/picture/常见k线走势.png)

步骤：

* 标记 市场周期：突破、窄通道、宽通道、震荡区间，寻找他们之间转变的信号，思考该如何交易
* 标记熟悉的结构或形态：双顶/底、楔形顶/底、头肩、三角形、圆弧底 等等
* 标记特殊的K线：大阳/阴线、孕线、吞噬线、ii、ioi 等等，标记信号k线/入场k线，Measured Move
* 像实盘一样思考：用多少仓位？止损、止盈的位置？出现哪些信号说明这笔交易的前提不再成立？
